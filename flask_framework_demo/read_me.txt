Project compiles flask template into a webpage. Comments remove Raspberry Pi imports not included in demo.

How to run:
	Open terminate and navigate to folder (flask_framework_demo)
	In folder run command "set FLASK_APP=demoTest.py" to compile. If running Mac use keyword "export" instead "set." Generates a cache folder to hold data.
	Run compiled data with command "flask run --host=0.0.0.0"
	After establishing connection, open browser to localhost:[port number]
